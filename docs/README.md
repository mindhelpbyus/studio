# Venture Healthcare Platform Documentation

## Component Library

### Venture Design System

The Venture design system provides a comprehensive set of reusable components:

#### Layout Components
- **VentureHero** - Hero sections with geometric background patterns
- **VentureCard** - Content card containers

#### Form Components
- **VentureSelect** - Single selection dropdown
- **VentureMultiSelect** - Multiple selections with checkboxes
- **VentureRadioSelect** - Single selection with radio buttons
- **VentureAvatarSelect** - User selection with avatars
- **VentureDetailedSelect** - Rich options with descriptions
- **VentureDatePicker** - Date selection with calendar
- **VentureCheckbox** - Boolean checkbox
- **VentureRadio** - Radio button
- **VentureInput** - Text input fields
- **VentureButton** - Action buttons

#### Data Display Components
- **VentureTable** - Data table with sorting
- **VentureDataTable** - Table with header and actions
- **VentureModal** - Modal dialogs
- **VentureConfirmModal** - Confirmation dialogs

## Color System

All components use semantic color tokens:
- `Color-Tokens-Background-Primary`
- `Color-Tokens-Background-Secondary`
- `Color-Tokens-Content-Dark-Primary`
- `Color-Tokens-Content-Dark-Secondary`
- `Color-Tokens-Content-Dark-Tertiary`
- `Color-Tokens-Interaction-Primary-Base`
- `Color-Tokens-Interaction-Secondary-Base`
- `Color-Tokens-Border-Primary`
- `Color-Tokens-Border-Secondary`

## Typography

All text uses the Inter font family with proper weights and sizes defined in the design system.
